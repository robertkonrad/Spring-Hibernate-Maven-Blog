# Blog

Simple blog application.

## Used technologies

* Spring Framework
* SpringBoot
* Hibernate
* Maven
* Thymeleaf

## How to run

Connect to database by changing url and login data in /src/main/resources/application.properties file.

Open terminal in project folder and use commands:
```
    ./mvnw clean install
```

    and

```
    ./mvnw spring-boot:run
```
